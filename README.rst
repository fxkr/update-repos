update-repos
============

Updates all repositories in a directory.

Usage::

  update-repos <directory>

Supported VCS:

* git (http://git-scm.com/)
* git-svn (part of git)
* svn (https://subversion.apache.org/)
* bzr (http://bazaar.canonical.com/)
* hg (http://mercurial.selenic.com/)
